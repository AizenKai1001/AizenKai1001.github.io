Xerox Bot

A feature-rich Discord bot with social interactions, moderation tools, and more

Discord.py Python v4.2

Overview

Xerox Bot is a comprehensive Discord bot built with discord.py, offering a wide range of features for community management, social interactions, and utility functions. Version 4.2 brings enhanced functionality and improved user experience.

Social Interactions

  • Profile system with XP and levels
  • Reputation system
  • Marriage and family relationships
  • Fun interaction commands
  • Daily rewards and streaks
  • Custom status settings
  • Server leaderboards

Moderation

  • Timeout management
  • Temporary bans
  • Advanced role management
  • Detailed logging system

Utility

  • Reminder system
  • Scheduled announcements
  • Help command with categories
  • Role information lookup

Games

  • Valorant agent images
  • Reddit gaming content

Command Categories

General

  • /help - View commands and usage
  • /hello - Get a friendly greeting

Social

  • /profile - View/customize your profile
  • /reputation - Give reputation points
  • /leaderboard - View server rankings
  • /marry - Propose to another user
  • /divorce - End your marriage
  • /family - View your family tree
  • /status - Set custom status
  • /daily - Claim daily rewards

Fun

  • /slap - Playfully slap someone
  • /kiss - Give someone a kiss

Utility

  • /remind - Set reminders
  • /list_reminders - View your reminders
  • /cancel_reminder - Cancel a reminder
  • /schedule_announcement - Schedule server announcements
  • /list_announcements - View scheduled announcements
  • /cancel_announcement - Cancel an announcement

Moderation

  • /timeout - Temporarily mute a user
  • /unmute - Remove a timeout
  • /tempban - Temporarily ban a user

Roles

  • /roleinfo - View role details
  • /temprole - Assign temporary roles

Games

  • /valagent - Get Valorant agent images
